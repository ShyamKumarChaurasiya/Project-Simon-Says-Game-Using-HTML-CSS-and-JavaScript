# Project-Simon-Says-Game-Using-HTML-CSS-and-JavaScript
A classic Simon Says memory game built using HTML, CSS, and JavaScript, where players repeat an increasingly complex sequence of colors and sounds. The project demonstrates DOM manipulation, event handling, and game logic implementation while delivering an interactive and engaging user experience.
